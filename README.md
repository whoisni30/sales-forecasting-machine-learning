# sales-forecasting-machine-learning
End-to-end machine learning project for sales forecasting using Python, Random Forest Regression, and business analytics.

# 📈 Sales Forecasting using Machine Learning

## 📌 Project Overview

This project focuses on forecasting product sales using Machine Learning techniques. It includes data preprocessing, feature engineering, exploratory data analysis (EDA), visualization, and predictive modeling to help businesses understand sales trends and make data-driven decisions.

---

## 🎯 Objectives

- Analyze historical sales data.
- Perform data cleaning and preprocessing.
- Engineer time-based features.
- Visualize sales trends and business insights.
- Build a Machine Learning model for sales forecasting.
- Evaluate model performance using regression metrics.

---

## 📂 Dataset

The project uses two datasets:

- **train.csv** – Superstore Sales Dataset
- **vgsales.csv** – Video Game Sales Dataset

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

---

## 📊 Exploratory Data Analysis

The following visualizations were created:

- Orders by Year
- Monthly Sales Trend
- Orders by Category
- Sales by Region
- Shipping Mode Distribution
- Customer Segment Distribution
- Top 10 States by Sales
- Top 10 Products by Sales
- Actual vs Predicted Sales

---

## 🤖 Machine Learning Model

**Random Forest Regressor**

The model was trained to predict sales using time-based and shipping-related features extracted from the historical sales data.

---

## 📈 Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Open `analysis.ipynb` in Jupyter Notebook or Google Colab.
4. Upload the required datasets (`train.csv` and `vgsales.csv`).
5. Run all cells sequentially.

## 💡 Key Insights

- Sales show seasonal variations across different months and years.
- Consumer customers generate the highest number of orders.
- Standard Class is the most frequently used shipping mode.
- Regional and category analysis provides valuable business insights.
- Machine Learning can support sales forecasting and business planning.
